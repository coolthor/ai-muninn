## 情境

你打開 AI，但網路上看到三個名字：ChatGPT、Claude、Gemini。

別人說 ChatGPT 最好、有人推 Claude、Google 廣告又一直推 Gemini。**選哪個？三個都付？三個都裝？**

不用。三個各有強項，**用對一個就夠**。

---

## 三個 AI，一句話分辨

### ChatGPT — 最通用、外掛最多

**你是誰會用**：第一次接觸 AI、不確定自己要什麼、想試各種功能（畫圖、寫程式、查資料、整理會議⋯⋯）的人。

**強的地方**：
- 外掛跟工具最多（內建畫圖、Data Analysis 跑程式跟分析資料、Custom GPTs）
- 用戶量最大、教學資源最多
- 跟 Apple Intelligence 整合（iPhone Siri 可選擇接 ChatGPT），Microsoft 365 Copilot 也用 OpenAI 模型

**弱的地方**：寫長文容易跑題、有時會自信地胡說（hallucination）。

### Claude — 寫長文 / 看大段文件最強

**你是誰會用**：要寫部落格、報告、論文；要分析合約、論文、長 email；做翻譯校稿。

**強的地方**：
- 付費版一次能讀 20 萬 token（約 500 頁文件）；Claude Code 部分模型有更大的 context
- 寫出來的東西像人話、邏輯連貫
- 寫長文 / 看大段資料的口碑最好

**弱的地方**：沒有官方畫圖、即時搜尋功能較陽春、外掛少。

### Gemini — Google 整合最好、即時資訊最快

**你是誰會用**：Gmail / Google Docs / Google Calendar 是日常工作主力；要查最新新聞、股價、即時資訊。

**強的地方**：
- 直接整合 Google Workspace（在 Gmail 裡按一下總結整封信）
- 即時搜尋網路有 Google 搜尋背書，新事件抓得快
- 免費額度也算大方（隨時間調整）

**弱的地方**：寫作風格較保守、寫程式比 Claude / GPT 弱一點。

## 30 秒決策流程

```
你日常工作主要是 Gmail / Docs / Sheets？
  → 是 → Gemini（整合便利贏）
  → 否 → 下一題

你要做的最重要的一件事是寫長文或看大量文字？
  → 是 → Claude
  → 否 → 下一題

預設 → ChatGPT（最通用、犯錯成本最低）
```

---

## 一個常見錯誤：三個都付

每個 Pro 月費 $20，三個 = 一個月 $60。**多數人用不到。** 日常先付一個，半年後看需求再決定要不要加。

如果硬要三個 — 通常是因為：寫作多用 Claude、查資料多用 Gemini、隨手用 ChatGPT。**確定有這三種需求才加碼。**

## 直接抄的決策提示

下次再有人問你「我該用哪個 AI」，貼這段：

```
看你最常做什麼事：
- 不知道要做什麼 / 各種功能想試 → ChatGPT
- 寫長文、看大段文件 → Claude
- 用 Gmail / Google Workspace、要查最新資訊 → Gemini

先付一個（每月 $20），用一陣子有不夠的需求再加。
```
